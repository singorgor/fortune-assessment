# 2026丙午·命盘运势推演录

基于传统八字智慧，为你解读2026年运势趋势的Next.js应用。

## 功能特点

- 🎯 **精准分析**：基于传统四柱八字子平法，分析个人命盘
- 🔥 **流年解读**：专门针对2026丙午火年的运势分析
- 📊 **五大领域**：事业、财运、婚恋、健康、人际全面覆盖
- 📅 **月度指引**：12个月逐月节气运势提示
- 💡 **个性化建议**：结合用户处境选择，提供定制化行动指南
- 🔒 **隐私保护**：纯前端计算，数据不上传
- 📱 **PWA支持**：可安装到手机，离线使用

## 技术栈

- **框架**：Next.js 15 (App Router)
- **样式**：Tailwind CSS + shadcn/ui
- **存储**：IndexedDB（本地）
- **语言**：TypeScript

## 核心算法

### 四柱计算
- 年柱：以立春为界，天干地支纪年
- 月柱：按节气分月，非农历初一
- 日柱：公历日期转干支纪日
- 时柱：十二时辰，每时辰两小时

### 五行分析
- 统计四柱干支五行力量
- 分析旺衰状态（偏强/偏弱/均衡）
- 判断用神忌神

### 十神关系
- 以日主为中心分析其他干支关系
- 包括比劫、食伤、财星、官杀、印星
